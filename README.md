# README
My dotfiles.

Install using `install.sh`

## Requirements
global
  * `git`
  * `vim` (>8.0 +python3)
  * `tmux`
  * `openssl`

vim
  * `python3`
  * `cmake`

bash
  * `mdlive` alias:
    * darwin
      * `fswatch`, `brew install fswatch`
      * `mdcat`, `brew install mdcat`
    * linux
      * `mdless`, `gem install mdless`
      * `inotifywatch`, `apt install inotify-tools`
